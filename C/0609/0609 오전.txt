따라서 10000000 비트를 사용하면 원래는 보수형태를 취해서  값이 나오지만 
%u를 사용하면 보수로 취하지 않고 맨앞의 부호비트를 일반 비트로 생각해 값이 나온다.

@조건문
if문에는 반드시 else문이 필요하다.
else if문 작성시 반드시 else가 필요하다. 현재 else문이 없어도 오류는 나지 않지만 버전에 따라 다름
if은 단독으로도 사용할 수 있다.
goto문은 사용하지 않는다라고 배우지만, 현업에서는 생각보다 많이 사용한다.
현업에서 고객이 짜둔 코드를 전부가 아닌 일부만 사용하라고 할 때 기존에 짜둔것을 버리기엔 아깝기때문에
goto문을 통해 실행 안할부분을 넘긴다. 아니면 if문에 조건을 0으로 만들어서 실행 안시킬 부분을 추가한다.
짜둔 코드의 일부만 실행하고 나머지는 실행시키지 않을 때 방법

1. 실행 안하는 부분 삭제
2. 주석
3. if문 조건을 false로 만듦 
-------------------------------------------------------1~3번은 빌드는 된다.
4. #if 0 ~ #endif 사용 (빌드조차 안시킴, 단 0 자리에 0이 아닌수를 사용하면 빌드가 된다.) 

#이 붙은 if문과 일반 if문의 차이는 컴파일을 하냐 안하냐이다. #이 붙은 if문은 조건이 거짓일 경우 컴파일 안함
