자바스크립트의 prototype은 함수의 원형을 찾아주는 예약어이다.
자바스크립트에서 객체의 속성을 추가할 때 추가할 부분을 만들어 값을 넣거나
prototype을 이용해서 만들어준다 프로토타입은 원형을 수정할 수 없을 때 사용할 수 있는 것이다.
ex)
var vv = new object()
vv.name = "asdf"
vv.age = 10
과 
vv.prototype.add = "sadfqwerq" 처럼 객체의 속성을 추가할 수 있다. 이 때 변수뿐만 아니라 함수도 가능
