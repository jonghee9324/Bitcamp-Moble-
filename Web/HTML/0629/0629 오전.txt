@ftp

@http 
무상태연결형

@URL
http     :     // www.naver.com                               : 80              /index.html
-----	     ---------------------		      -------             --------------
프로토콜        ip주소를 알기쉬운 주소로 갖음	       포트번호        문서 위치, 경로

naver.com이라 쓰면 http://www를 추가해줌
naver.com만 쓰면 도메인 서버에 naver.com이라는 홈페이지 주소가 있는지 물어봄
80번 포트는 생략가능(서비스 받는 포트번호)
서비스 종류마다 기본적으로 갖는 포트번호가 다름(ftp의 경우21, http는 80)
/ 앞에까진 프로토콜, 컴퓨터주소, 포트번호, /뒤가 파일을 가져올 경로

클라이언트의 파일을 가져오는 경우
localhost엔 클라이언트 pc의 호스트네임이 들어감
