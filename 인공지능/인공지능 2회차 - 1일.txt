@numpy
0차원 (값 : 스칼라)
1차원 (리스트 : 벡터)
2차원 (행렬)
3차원 (행렬 + 폭 : 텐서)

arange(숫자) : 숫자만큼의 1차원 행렬을 만들어줌

arange(숫자).reshape(만들 행렬) : arange로 만든 행렬을 만들행렬로 변경해줌
ex ) arange(6).reshape(2,3) : 0~5로 이루어진 리스트를 2행3열로 만들어줌 ( 단, reshape의 범위로 적은 숫자의 곱은 arange에 적은 숫자와 같아야함)



@@@@ 표기법

Camel(낙타) 표기법 : 처음 소문자로 시작해서 첫글자를 대문자로 표시 (camelCase) 
Pascal 표기법 : 첫글자만 대문자로 표시, 처음도 대문자 (PascalCase)
snake 표기법 :  단어를 밑줄(_)로 연결, 다 소문자 (snake_case)
헝가리언 표기법 : 변수명 사용시 자료형 구분을 위한 접두사를 사용
n : 정수
i : 인덱스변수 (정수)
f : 실수형변수
b : 참, 거짓을 나타내는 변수
dw : DWORD 단위의 변수 
g_ : 글로벌 변수(전역 변수)
m_ : 멤버변수

주로 헝가리언 표기법과 snake or camel or pascal
